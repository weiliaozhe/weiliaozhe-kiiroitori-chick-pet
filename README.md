# Kiiroitori chick pet

English: A Codex pet bundle for `Kiiroitori chick pet`.

中文：这是一个给 Codex 用的 `Kiiroitori chick pet` 宠物包，也就是我说的“鼻孔鸡”。

## Files / 文件

- `kiiroitori-chick-pet/pet.json`
- `kiiroitori-chick-pet/spritesheet.webp`
- `preview/contact-sheet.png`

## Install / 安装

1. Copy the `kiiroitori-chick-pet` folder into your local Codex pets directory.  
   把 `kiiroitori-chick-pet` 文件夹复制到你本地的 Codex 宠物目录。

   - Windows: `C:\Users\<you>\.codex\pets\`
   - macOS/Linux: `~/.codex/pets/`

2. Restart Codex if it is already open.  
   如果 Codex 已经打开，重启一次。

3. Select `Kiiroitori chick pet` from your available pets.  
   然后在可用宠物列表里选择 `Kiiroitori chick pet`。
